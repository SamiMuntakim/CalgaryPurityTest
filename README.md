# Calgary Purity Test

A purity test for University of Calgary students. Find out how "pure" your Calgary student experience has been.

Take the test -> calgarypuritytest.com

## About

The Calgary Purity Test measures your exposure to Calgary student life with 100 questions covering everything from commuter struggles to academic despair to late nights at The Den.

Based on the classic Rice Purity Test format, but specifically tailored for UCalgary and Calgary experiences.

## Features

- 100 Calgary/UofC-specific questions
- Score from 0-100 (lower = less pure)
- Personality type based on your answers
- Share your results on social media

## Tech Stack

- React + Vite
- Deployed on Vercel

## Run Locally

```bash
npm install
npm run dev
```
